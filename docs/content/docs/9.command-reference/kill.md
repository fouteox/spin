# spin kill
::lead-p
Send `SIGKILL` to all containers. See the documentation on [docker kill](https://docs.docker.com/engine/reference/commandline/kill/) for more advanced usage.
::

## Usage
::code-panel
---
label: Usage for "spin kill"
---
```bash
spin kill
```
